# Launch Studio Textual v1

Launch Studio Textual v1 is a local-only static launch readiness cockpit for a founder reviewing whether to launch. It covers lifecycle status, launch risks, QA state, SEO readiness, and launch boundaries without requiring a backend.

## Local run

1. Open this project folder.
2. Open `index.html` in a browser from the folder view, or serve the folder with any local static-file tool you already use.
3. Review the checklist, write a risk note, and choose a founder launch stance.

The app stores checklist state and notes in browser local storage only.

## Hard boundaries

- No deployment.
- No credentials.
- No public messages.
- No paid spend.
- No external API calls.
- No analytics beacons.
- No backend dependency.

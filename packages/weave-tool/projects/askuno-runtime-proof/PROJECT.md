# Askuno Runtime Proof

Askuno Runtime Proof is the first admitted WEAVE worked example. It records how
the wrapper guides an execution runtime through intent, research, selection,
planning, engineering, QA, KPI setup, marketing, and iteration.

The project must proceed through lifecycle gates. Engineering work depends on
Research admission. QA depends on Engineering evidence. Iteration depends on
KPI setup, marketing, and analytics evidence.

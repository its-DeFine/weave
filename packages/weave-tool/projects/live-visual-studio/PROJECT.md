# Live Visual Studio

Live Visual Studio is the first admitted WEAVE application candidate. It gives
creators a browser-native realtime visual surface now while keeping every
primitive shaped for a later Livepeer adapter swap.

The project must proceed through lifecycle gates. Engineering work depends on
Research admission. QA depends on Engineering evidence. Iteration depends on
KPI or analytics evidence.

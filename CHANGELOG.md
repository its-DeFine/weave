# Changelog

## v0.1.0 - 2026-06-22

Initial public COS WEAVE release.

- Defines the prompt-first COS WEAVE startup path for Codex threads.
- Provides the visible file skeleton under `runs/cos-weave-home/`.
- Supports multiple app folders with intent, todos, lifecycle, worker packets,
  proof, blockers, review, and readback state.
- Adds lifecycle procedures from intent through analysis.
- Adds docs-currentness, package, safety, and public-safe validation checks.
- Includes user-facing visual flow and lifecycle diagrams.
- Removes old default product surfaces from the current public path.

Non-claims:

- v0.1.0 does not prove live worker spawning.
- v0.1.0 does not mutate trackers or external systems by default.
- v0.1.0 does not deploy, send public messages, use credentials, or perform
  paid actions unless a later authorized lifecycle step explicitly does so.
